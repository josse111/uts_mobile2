# Smart To-Do List 📝✨

**Smart To-Do List** adalah aplikasi manajemen tugas modern yang dirancang untuk meningkatkan produktivitas dengan antarmuka yang intuitif. Aplikasi ini membantu pengguna mengatur jadwal harian, memprioritaskan tugas, dan memastikan tidak ada pekerjaan yang terlewatkan.

## 🚀 Fitur Utama

*   **Manajemen Tugas:** Tambah, edit, dan hapus tugas dengan mudah.
*   **Prioritas & Kategori:** Tandai tugas berdasarkan tingkat urgensi (Low, Medium, High).
*   **Status Tugas:** Pantau progres tugas yang sedang berjalan atau yang sudah selesai.
*   **Desain Responsif:** Tampilan yang optimal baik di perangkat mobile maupun desktop.
*   **Penyimpanan Lokal/Cloud:** (Sesuaikan: misal menggunakan SQLite, Room, atau Firebase).

## ⚙️ Instalasi

Ikuti langkah-langkah berikut untuk menjalankan proyek ini secara lokal:

1.  **Clone repositori ini**
    ```bash
    git clone https://github.com/username-anda/smart-todo-list.git
    ```
2.  **Masuk ke direktori proyek**
    ```bash
    cd smart-todo-list
    ```
3.  **Build & Run**
    *   Jika menggunakan **Android Studio**: Buka proyek dan klik tombol *Run*.
    *   Jika menggunakan **Web/PHP**: Pindahkan folder ke `htdocs` dan jalankan server lokal.
